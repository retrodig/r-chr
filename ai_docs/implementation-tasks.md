# 実装タスク

## 実装済み機能

### ファイル入出力
- [x] `.nes` ファイルの読み込み・保存（iNES フォーマット）
- [x] `.bin` ファイルの読み込み・保存（生 CHR バイナリ）
- [x] `.zip` ファイルの読み込み（内部の .nes を展開）
- [x] `.nes` / `.bin` / `.zip` のドラッグ & ドロップ対応
- [x] `.png` / `.bmp` のインポート（CHR カラーインデックス変換）
- [x] `.png` / `.bmp` のドラッグ & ドロップ対応
- [x] `.pal` ファイルの読み込み（マスターパレット差し替え）
- [x] `.dat` ファイルの読み込み・保存（DAT パレット）
- [x] 未保存変更の確認ダイアログ（保存 / 破棄 / キャンセル）
- [x] **新規作成**（空 0x4000 バイトの CHR データを newfile として生成）

### CHR 表示・操作
- [x] CHR 全体ビュー（128px 幅、16 タイル/行）
- [x] バンク切り替え（4KB 単位）
- [x] 選択ブロックサイズ切り替え（8 / 16 / 32 / 64 / 128 px）
- [x] アドレスジャンプ（16 進数入力）
- [x] テクスチャキャッシュ（texture_dirty フラグによる効率的な再描画）

### ドット編集
- [x] 8×8 ピクセルグリッドエディタ
- [x] 描画色選択（0〜3 の 4 色）
- [x] スポイトツール
- [x] アンドゥ（Cmd+Z / Ctrl+Z）
- [x] 矢印キーによるタイル選択移動・スクロール追従

### パレット
- [x] DAT パレット（4 セット × 4 色）の表示・編集
- [x] NES マスターパレット 64 色ピッカー
- [x] デフォルトパレットへのリセット

### UI / UX
- [x] ダークモード / ライトモード切り替え
- [x] macOS ネイティブメニューバー（muda）
- [x] Windows / Linux egui メニューバー
- [x] 日本語フォント（Noto Sans JP）対応
- [x] ウィンドウアイコン設定
- [x] macOS ウィンドウ枠ダークモード（NSAppearance）
- [x] 起動時デフォルト CHR データ表示（R-CHR ロゴ）
- [x] About ダイアログ

### コード構成
- [x] `io/` `model/` `editor/` によるモジュール整理
- [x] `editor/keyboard.rs` へのキーボード処理の分離
- [x] `editor/mac/menu_events.rs` への macOS ネイティブメニューイベント分離
- [x] `editor/mac/menu_bar.rs` への egui メニューバー分離
- [x] `editor/file_ops.rs` へのファイル操作分離
